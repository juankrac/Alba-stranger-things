# 🌀 Stranger Things - Página Web del Upside Down

Una página web interactiva y temática dedicada a la serie **Stranger Things**, decorada con los colores y la estética del Upside Down.

## ✨ Características

### 🎬 Cuatro Secciones Principales:

1. **Inicio** 👻
   - Bienvenida al Upside Down
   - Portal animado interactivo
   - Introducción a la serie

2. **Personajes** 👥
   - Galería de personajes principales
   - Descripciones de cada personaje
   - Tarjetas interactivas con efectos hover

3. **Curiosidades** 📚
   - Datos fascinantes sobre la serie
   - Información sobre la producción
   - Datos de éxito y reconocimiento

4. **Creatividad** 🎨
   - Sección de comunidad
   - Inspiración artística
   - Formulario de contacto para compartir creaciones

## 🎨 Diseño Visual

- **Paleta de Colores**: Púrpura (#8b00ff) y Rosa Neón (#ff0078)
- **Efectos Especiales**:
  - Gradientes animados
  - Partículas flotantes
  - Efectos de glow y brillo
  - Animaciones suaves en scroll
  - Portal giratorio interactivo

- **Tipografía**: Monospace (Courier New) para una sensación retro
- **Fondo**: Animación dinámica que simula la esencia del Upside Down

## 🛠️ Tecnologías Utilizadas

- **HTML5**: Estructura semántica
- **CSS3**: Estilos avanzados con animaciones y efectos
- **JavaScript**: Interactividad y efectos dinámicos

## 🚀 Características Interactivas

### Efectos de Navegación
- Menú sticky con efectos glow
- Links suavizados con scroll automático
- Indicador de sección activa

### Animaciones
- Aparición progresiva de elementos
- Efecto float en iconos
- Partículas que siguen el cursor
- Glitch effect en el logo

### Formulario
- Validación de campos
- Efectos visuales en enfoque
- Mensaje de confirmación

### Easter Egg 🥚
- Presiona `Ctrl + D` para descubrir un mensaje secreto

## 💡 Consejos de Uso

1. Navega por las secciones usando el menú principal
2. Interactúa con las tarjetas de personajes y curiosidades
3. Envía tu creación de Stranger Things a través del formulario
4. Busca el easter egg escondido
5. Observa los efectos animados mientras te desplazas

## 📱 Diseño Responsive

## 🎯 Objetivos del Proyecto

Este proyecto busca:
- Celebrar la serie Stranger Things
- Crear una experiencia visual inmersiva del Upside Down
- Demostrar técnicas modernas de desarrollo web
- Fomentar la creatividad de la comunidad de fans
